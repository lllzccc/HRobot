# HRobot v0.1.3

本次版本集中更新桌面端发布能力、个性头像和近期前端体验。

## 新增

- 新增 24 个可选个性头像，可在设置中选择，并同步显示在首页与 AI 问答助手。
- 新增原生 macOS 应用包，分别支持 Apple 芯片和 Intel 芯片 Mac。
- 新增 GitHub Actions 自动构建：发布版本时同步生成 Windows 安装包与两个 macOS DMG。
- macOS 应用支持在设置中检查更新并点击覆盖安装。

## 优化

- 优化首页、报告生成、报告详情和设置页的布局、间距、对齐与交互反馈。
- 优化密集 HR 工作台的信息层级与表格可读性。
- macOS 用户数据独立保存在 `~/Library/Application Support/Hrobot`，更新应用不会覆盖本地数据和个人设置。
- Windows 更新继续保留安装目录中已有的 `data` 数据。

## 安全与打包

- 安装包仅包含程序、通用静态资源和空数据目录骨架。
- 不包含 HR 源数据、权限数据、API Key、上传文件、生成报告、日志和本地缓存。
- 发布素材采用 HRobot 白名单，已排除三七互娱、三七游戏及相关旧品牌图片。

## macOS 首次打开提示

打开 DMG 后，请先把 Hrobot 拖到 `Applications` 文件夹，再从“应用程序”中启动；不要直接在 DMG 中长期运行。

当前版本尚未使用 Apple Developer 证书公证。首次打开时如系统提示无法验证开发者，请右键 Hrobot 应用并选择“打开”。
